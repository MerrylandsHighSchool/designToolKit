MHS-Devs Design Toolkit

A virtual toolkit of links and resources to help you increase the quality of your designs

  • Google Keep

    A simple to do list tool synced to Google Account

  • Asana

    A more powerful to-do list with depentent tasks, timeline and task allocation

  • Team Gantt Recommended!

    A powerful and easy to use online Gantt chart tool. What is a Gantt Chart? Read this to learn more about Gantt Charts

  • Gantt Project

    A user friendly, offline Gantt Chart tool. Download the no-installer version here to run on any school computer

  • Monday

    A powerful (and complex) project management tool, incorporating tasks, timelines, resource management and Gantt charts. Best for team and/or complex projects.MHS-Devs has a premium account, see Mr Jones.

  • Adobe Color

    A powerful colour palette tool that integrates with Adobe products. Note; you need to sign in with your Adobe ID.

  • Random Color Palette

    Uses AI and deep learning to create random colour palettes.

    • Color Palette Generator

    Create a colour palette from a stimulus image.

  • 100 color combinations

    100 themed colour schemes for use in any visual design (games, apps, websites, documents, etc).

  • 50 Shades of Grey

    Advice and colour schemes for using shades of grey in design.

  • Aspect ratio calculator

    Simple calculator for resising images without losing the aspect ration

  • Tiny PNG

    Optimise any image (png,jpeg, etc) for websites or applications

  • Font Awesome

    Vector icons and social logos on your website as a font. Use Flaticon to download icons as a PNG or SVG to use in graphic or document deisgn projects. Font Awesome is inbuilt to Bootstrap

  • Flaticon

    Extensive vector icon repository, download as SVG or PNG. Flaticon is another similar alternative.

  • Google Fonts

    Comprehensive repository and CDN of fonts and web fonts
