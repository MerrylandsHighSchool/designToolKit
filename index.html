MHS-Devs Design Toolkit

A virtual toolkit of links and resources to help you increase the quality of your designs

  • Adobe Software at home

    1. Go to the Student BYOD Software Hub
    2. Register using your @education.nsw.gov.au student email
    3. Choose Adobe CC and click through the checkout process
    4. Check your school emails for confirmation and link to 'redeem' your code or go back to your account in the BYOD Student Hub
    5. Copy your redemption code
    6. Click on 'Redeem your code'
    7. Paste the redemption code in the space provided
    8. Follow the link and install the Adobe Creative Cloud Applications Manager
    9. Now you can install the desktops apps you wish.
    10. On your mobile, install Adobe mobile apps as you wish and log in using your Adobe account (that you made with @education.nsw.gov.au email)

  • Microsoft Software at home

    1. Go to the Student BYOD Software Hub
    2. Register using your @education.nsw.gov.au student email
    3. Choose Microsoft, Microsft Office and click through the checkout process
    4. At the end of the checkout process, you will be taken to a page with a license code and a link to download the software
    5. Print this page as a PDF and save the file to your school Google Drive account. You can only do this once and once the code is wiped you can not get another license code
    6. Download the software (skip this step if your computer already has a trial version installed)
    7. During the setup process enter the license code provided in step 4
